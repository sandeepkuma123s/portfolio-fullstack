## Features

- Responsive portfolio UI
- Animated sections
- Projects loaded from MongoDB
- Contact form connected to backend
- Resume download option
- Full-stack deployment

## Tech Stack

### Frontend

- CSS
- Framer Motion
- Axios

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
